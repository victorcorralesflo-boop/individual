# Conclusión personal

Durante el desarrollo de este proyecto pude aplicar conceptos importantes de programación orientada a objetos en C++. Aprendí a organizar mejor un programa utilizando clases con responsabilidades específicas, lo que hace que el código sea más claro, ordenado y fácil de modificar.

También comprendí mejor el uso de herencia, polimorfismo, clases abstractas, sobrecarga y sobreescritura de métodos. Estos conceptos me ayudaron a diseñar una estructura más flexible, especialmente al representar un modelo de inteligencia artificial como una clase abstracta y después implementar una versión específica para Ollama.

Además, reforcé la importancia de validar las entradas del usuario para evitar errores durante la ejecución. Considero que esta actividad me ayudó a mejorar mis habilidades de análisis, diseño e implementación, además de darme una mejor idea de cómo conectar un programa en C++ con herramientas modernas de inteligencia artificial.
