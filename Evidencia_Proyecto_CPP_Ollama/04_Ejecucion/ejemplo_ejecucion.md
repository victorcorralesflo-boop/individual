# Ejemplo de ejecución

Ejemplo esperado al ejecutar el programa:

```txt
======================================
     Chatbot en C++ usando Ollama
======================================
Escribe tu nombre: Victor

Hola Victor.
Escribe un mensaje para la IA.
Escribe 'salir' para terminar.

Victor: Que es C++
IA: Respuesta generada por gemma: Recibí tu mensaje: "Que es C++"

Victor: salir
Chat finalizado correctamente.

----- Historial de conversación -----
Usuario: Que es C++
IA: Respuesta generada por gemma: Recibí tu mensaje: "Que es C++"
------------------------------------
```

En esta ejecución se observa que el sistema solicita el nombre del usuario, valida los mensajes escritos, responde al usuario y permite terminar correctamente con el comando `salir`.

En la carpeta `capturas` se deben agregar imágenes de la ejecución real desde la terminal.
