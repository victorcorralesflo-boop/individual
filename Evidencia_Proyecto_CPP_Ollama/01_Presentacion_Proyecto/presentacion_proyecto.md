# Presentación del proyecto

El proyecto consiste en el desarrollo de una aplicación en C++ que permite la interacción entre un usuario y un modelo de inteligencia artificial ejecutado localmente mediante Ollama. El sistema funciona como un chatbot en consola, donde el usuario escribe un mensaje, el programa valida la entrada, construye una solicitud y obtiene una respuesta del modelo configurado.

El objetivo principal es crear una solución que conecte un programa desarrollado en C++ con un modelo de lenguaje local, aplicando conceptos de programación orientada a objetos como clases, objetos, herencia, polimorfismo, clases abstractas, sobreescritura de métodos, sobrecarga de métodos y modificadores de acceso.

El sistema busca resolver la necesidad de contar con una interfaz sencilla para comunicarse con un modelo de inteligencia artificial local sin depender directamente de una plataforma externa.
